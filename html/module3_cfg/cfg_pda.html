Equivalence Theorem

Theorem (Sipser 2.20)
A language is context-free if and only if some pushdown automaton recognizes it.

i.e.  L is CFL ⟺ ∃ PDA P such that L(P) = L
Choose Direction
Key Insight: A PDA can simulate leftmost derivation of a CFG using its stack. Conversely, any PDA can be converted into a CFG whose productions mirror the PDA's transitions.
Class Hierarchy
Regular ⊂ CFL ⊂ CSL ⊂ RE
DFA ⊂ PDA = CFG ⊂ LBA ⊂ TM

Step-by-Step Proof

Given CFG G = (V, Σ, R, S), construct PDA P that accepts L(G) by simulating leftmost derivation on its stack. Click each step to expand.
STEP 1Initialize the Stack
Push the bottom-of-stack marker $ then the start variable S. The PDA starts in state q_start and immediately moves to q_loop:
  • δ(q_start, ε, ε) = {(q_loop, S$)}
STEP 2Expand Non-terminals (Predict)
Whenever the top of the stack is a non-terminal A ∈ V, nondeterministically replace it with any RHS of a rule A → w:
  • For each rule A → w₁w₂…wₖ in R:
  • δ(q_loop, ε, A) = {(q_loop, w₁w₂…wₖ)}
This simulates a leftmost derivation step.
STEP 3Match Terminals (Scan)
Whenever the top of the stack is a terminal a ∈ Σ that matches the current input symbol, pop it and advance the head:
  • For each terminal a ∈ Σ:
  • δ(q_loop, a, a) = {(q_loop, ε)}
STEP 4Accept on Empty Stack
When the stack bottom marker $ is on top and all input is consumed, move to the accept state:
  • δ(q_loop, ε, $) = {(q_accept, ε)}
The resulting PDA P has states {q_start, q_loop, q_accept} and L(P) = L(G). ∎
Given PDA P, construct CFG G such that L(G) = L(P). Each PDA transition becomes a set of CFG productions. Click each step to expand.
STEP 1Normalize the PDA
First bring P to a standard form where:
  • It has a single start state q₀ and single accept state q_acc
  • It accepts only by empty stack
  • Each transition either pushes or pops exactly one symbol
STEP 2Introduce Variables A[pq]
Create a grammar variable A[pq] for every pair of states (p, q) in P. This variable generates all strings that take P from state p with empty stack to state q with empty stack:
  • Start symbol: S → A[q₀, q_acc]
STEP 3Add Productions (via-state split)
For every pair (p, q) and every intermediate state r:
  • A[pq] → A[pr] A[rq]
This captures strings that take P from p to some midpoint r, then from r to q.
STEP 4Add Productions (push-pop pairs)
For every push transition δ(p, a, ε) containing (r, t) and every pop transition δ(s, b, t) containing (q, ε):
  • A[pq] → a A[rs] b
Here a is pushed at p and t is popped at some later state s reaching q.
STEP 5Add ε-productions
For each state p:
  • A[pp] → ε
Represents staying in the same state without consuming input. The resulting CFG G has L(G) = L(P). ∎
