CFG → CNF 🔄 Module 3

Convert any Context-Free Grammar to Chomsky Normal Form through the 5 Sipser steps: START → DEL → UNIT → TERM → BIN. Each step is shown with the updated rule set.

Input Grammar

CNF Conversion Steps
STEP 0Original grammar
STARTAdd new start symbol S₀
DELEliminate ε-rules
UNITEliminate unit rules (A → B)
TERMIsolate terminals in long rules
BINBinarize rules with |RHS| ≥ 3
